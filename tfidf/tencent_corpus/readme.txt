====== DESCRIPTION ======

* Structure of this package:

	----release_data
		----readme.txt		present file
		----data.txt		82,010 annotated prompt-response pairs


* Structure of data.txt file
	Each line consists of three elements: prompt, response, and manual annotation (average ratings given by two annotators), which are splitted by \tab. 
	We use UTF-8 to encode the Chinese texts in each data file.
